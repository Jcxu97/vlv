Place **`ffmpeg.exe`** and **`ffprobe.exe`** here (same folder) to avoid downloading via static-ffmpeg/GitHub on first run.

Windows builds: [Gyan FFmpeg builds](https://www.gyan.dev/ffmpeg/builds/) → *ffmpeg-release-essentials.zip* → copy both from `bin/`.

These binaries are **not** committed (see root `.gitignore`).
